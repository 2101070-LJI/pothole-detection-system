## 🐳 컨테이너 구조

### 기존 구조 (docker-compose.yml)
1. **web-server** (Apache) - 프록시 서버
2. **dashboard** (Streamlit) - 대시보드
3. **ai-core** (AI Engine) - 비디오 처리
4. **db** (PostgreSQL) - 데이터베이스
5. **cloudflared** (Cloudflare Tunnel) - 외부 접근

### 최적화된 구조 (docker-compose-optimized.yml)
1. **app** (dashboard + ai-core 통합) - 대시보드 + AI 엔진
2. **db** (PostgreSQL) - 데이터베이스
3. **cloudflared** (Cloudflare Tunnel) - 외부 접근

## 🔧 주요 기능

### AI Core
- YOLOv8 포트홀 감지
- Depth Anything V2 깊이 검증
- NPU Worker 연동
- 자동 파인튜닝 (APScheduler)
- Kakao Map API 연동

### Dashboard
- Streamlit 대시보드
- 인증 시스템
- 통계 및 분석
- 비디오 결과 페이지
- 관리자 기능
- **Phi-3-mini-int4 챗봇** (OpenVINO NPU 가속)
- Gemini 요약 기능 (데이터 요약용, 선택사항)

### Database
- PostgreSQL
- Django ORM

## 🚀 실행 방법

### 기존 구조 (5컨테이너)
```powershell
docker-compose up -d
```

### 최적화된 구조 (3컨테이너)
```powershell
docker-compose -f docker-compose-optimized.yml up -d
```
