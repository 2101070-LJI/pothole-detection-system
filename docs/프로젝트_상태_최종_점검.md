## ⚠️ 주의 사항

### 1. Nginx (web-server) Health Check 실패

**현상**: 
- Nginx 컨테이너가 unhealthy 상태로 표시됨
- 하지만 대시보드는 직접 접근 가능 (포트 8501)

**원인**: 
- Nginx health check가 dashboard 컨테이너에 연결하지 못함
- 네트워크 연결 문제일 수 있음

**영향**: 
- 대시보드는 직접 접근 가능하므로 기능상 문제 없음
- http://localhost:8501로 접근하면 정상 동작

**해결 방법** (선택사항):
```powershell
# Nginx 재시작
docker-compose restart web-server

# 또는 Nginx 로그 확인
docker-compose logs web-server
```

### 2. NPU Worker 미실행

**현상**: 
- NPU Worker가 실행되지 않음 (포트 9001)

**조치 필요**: 
별도 PowerShell 터미널에서 실행:
```powershell
cd C:\Users\your_username\Desktop\test
.\start_npu_worker.ps1
```

**영향**: 
- 깊이 검증 기능이 동작하지 않음
- 포트홀 탐지는 가능하지만 깊이 검증은 불가
