# Deep-Guardian 2-Container 구조 기능 포함 상태

## ✅ 포함된 기능

### 1. 카카오 API (Kakao Map API)
- **상태**: ✅ 포함됨
- **위치**: `inference-container/main.py`
- **기능**:
  - 좌표 → 주소 변환 (`get_address_from_coords`)
  - 주변 시설물 검색 (`search_nearby_facilities`)
  - 위치 기반 위험도 계산
- **환경 변수**: `KAKAO_MAP_APP_KEY` (docker-compose.yml에 설정됨)
- **기본값**: `your_kakao_map_app_key_here`

### 2. 자동 파인튜닝 (Auto Fine-tuning)
- **상태**: ✅ 포함됨
- **위치**: `inference-container/main.py`
- **기능**:
  - APScheduler를 사용한 스케줄러
  - 매일 00시에 자동 체크
  - 100장 이상 새 이미지 수집 시 자동 파인튜닝 실행
  - 파인튜닝 데이터셋 자동 준비
- **설정**:
  - `finetune_threshold = 100` (100장 이상)
  - `setup_scheduler()`: 매일 00시 Cron 작업
  - `check_and_finetune()`: 조건 확인 및 파인튜닝 실행
  - `finetune_model()`: YOLOv8 모델 파인튜닝
- **의존성**: `APScheduler>=3.10.0` (requirements.txt에 포함)

## ⚠️ 부분 포함 / 별도 실행 기능

### 3. Phi-3-mini-int4 (SLM 챗봇)
- **상태**: ⚠️ 별도 worker로 존재 (컨테이너에 포함 안 됨)
- **위치**: 
  - `slm_npu_worker_phi3.py` - Phi-3 Worker (Windows Host)
  - `dashboard/slm_npu_chatbot.py` - 챗봇 클라이언트
- **기능**:
  - OpenVINO GenAI를 사용한 Phi-3 모델 실행
  - NPU 가속 지원
  - HTTP API 서비스
- **현재 실행 방식**:
  - Windows Host에서 별도로 실행 (`start_phi3_worker.ps1`)
  - 포트: 기본적으로 9002 (또는 설정에 따라)
- **모델 위치**: `models/llm/Phi-3-mini-int4/`

## 🔧 통합 옵션

### 옵션 1: Phi-3를 LAMP 컨테이너에 통합 (권장)

**장점**:
- 모든 기능이 컨테이너 내부에서 실행
- 관리가 용이
- 배포가 간단

**구현 방법**:
1. LAMP 컨테이너에 Phi-3 Worker 추가
2. Flask 앱에서 Phi-3 API 호출
3. 웹 대시보드에 챗봇 UI 추가

### 옵션 2: Phi-3를 별도 컨테이너로 유지

**장점**:
- 독립적인 스케일링 가능
- 리소스 분리

**구현 방법**:
1. `phi3-worker` 컨테이너 추가
2. docker-compose.yml에 서비스 추가
3. LAMP 컨테이너에서 HTTP로 호출

### 옵션 3: Windows Host에서 계속 실행 (현재 상태)

**장점**:
- NPU 직접 접근 가능
- 기존 설정 유지

**단점**:
- 컨테이너 외부 의존성
- 배포 복잡도 증가

## 🚀 Phi-3 통합 방법

### 방법 1: LAMP 컨테이너에 통합 (간단)

1. **LAMP 컨테이너 Dockerfile 수정**:
```dockerfile
# Phi-3 관련 패키지 추가
RUN pip3 install openvino-genai openvino
```

2. **Flask 앱에 Phi-3 엔드포인트 추가**:
```python
@app.route('/api/chat', methods=['POST'])
def chat():
    # Phi-3 Worker 호출 또는 직접 실행
    ...
```

3. **웹 UI에 챗봇 추가**:
```html
<!-- index.html에 챗봇 UI 추가 -->
```

### 방법 2: 별도 컨테이너 추가

1. **docker-compose.yml에 추가**:
```yaml
services:
  phi3-worker:
    build:
      context: ./phi3-worker
      dockerfile: Dockerfile
    container_name: deep-guardian-phi3
    ports:
      - "9002:9002"
    volumes:
      - ./models/llm:/app/models
    networks:
      - deep-guardian-network
```

2. **LAMP 컨테이너에서 호출**:
```python
PHI3_URL = "http://phi3-worker:9002/chat"
```
