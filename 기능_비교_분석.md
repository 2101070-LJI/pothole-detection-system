# 기존 프로젝트 vs 새로운 2-Container 구조 기능 비교

## ⚠️ 중요: 기능적으로 완전히 동일하지 않습니다

### ✅ 포함된 기능 (동일)

1. **AI 추론 (inference-container)**
   - ✅ YOLOv8 포트홀 탐지
   - ✅ NPU Worker 호출 (깊이 검증)
   - ✅ 카카오 API (좌표→주소, 주변 시설물 검색)
   - ✅ 자동 파인튜닝 (매일 00시, 100장 이상 시)
   - ✅ 데이터베이스 저장

2. **기본 웹 기능 (lamp-container)**
   - ✅ 포트홀 데이터 조회 API (`/api/potholes`)
   - ✅ 지도 시각화 (Leaflet)
   - ✅ 포트홀 목록 표시
   - ✅ Phi-3 챗봇 (`/api/chat`)
   - ✅ 정적 파일 서빙
