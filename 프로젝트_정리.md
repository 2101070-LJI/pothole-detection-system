# Deep-Guardian 프로젝트 정리 문서

## 📋 프로젝트 개요

**Deep-Guardian**은 도로 포트홀을 자동으로 탐지하고 검증하는 AI 기반 시스템입니다. 
- **YOLOv8**을 사용한 포트홀 탐지
- **Depth Anything V2** 모델을 사용한 깊이 검증
- **PostgreSQL** 데이터베이스에 결과 저장
- **Streamlit** 대시보드를 통한 시각화
- **Apache** 웹서버를 통한 외부 접근

---

## 🏗️ 시스템 아키텍처

```
[사용자 브라우저]
    ↓
[Apache 웹서버 (포트 80)] - Reverse Proxy
    ↓
[Streamlit 대시보드 (포트 8501)] - UI/시각화
    ↓
[PostgreSQL 데이터베이스 (포트 5432)] - 데이터 저장
    ↑
[AI Core 컨테이너] - 포트홀 탐지 및 검증
    ↓
[Windows Host NPU Worker (포트 9001)] - 깊이 검증 (OpenVINO)
```

## 🔄 처리 흐름

### 1. 비디오 입력 처리
```
비디오 파일 (test4.mp4 등)
    ↓
AI Core 컨테이너에서 프레임 추출
    ↓
YOLOv8로 포트홀 탐지
    ↓
포트홀 영역만 Crop
```

### 2. 깊이 검증
```
Crop된 포트홀 이미지
    ↓
NPU Worker (Windows Host)로 전송
    ↓
Depth Anything V2 모델로 깊이 추정
    ↓
depth_ratio 계산 및 검증
    ↓
검증 통과 시에만 다음 단계로
```

### 3. 데이터 저장
```
검증된 포트홀 데이터
    ↓
위치 정보 (GPS) 추출
    ↓
위험도 계산 (risk_zones.json 기반)
    ↓
PostgreSQL 데이터베이스 저장
```

### 4. 시각화
```
데이터베이스 조회
    ↓
Streamlit 대시보드에서 지도 표시
    ↓
Apache를 통해 외부 접근
```

## 📌 주의사항

1. **NPU Worker**는 Windows Host에서 실행되어야 함
2. **모델 파일** 경로 확인 필요
   - YOLOv8: `ai-core/models/best2.pt`

3. **GPU 설정**은 선택사항 (CPU로도 동작 가능)
4. **Cloudflare Tunnel**은 외부 접근 시에만 필요
