## 🔄 데이터 처리 흐름

```
[비디오 파일]
    │
    ▼
[Inference Container]
    │
    ├─→ [YOLOv8] → 포트홀 탐지
    │
    ├─→ [NPU Worker] → 깊이 검증
    │      │
    │      └─→ depth_ratio 계산
    │
    ├─→ [카카오 API] → 위치 정보 변환
    │
    └─→ [MySQL DB] → 데이터 저장
            │
            ▼
    [LAMP Container]
            │
            ├─→ [Flask API] → 데이터 조회
            │
            └─→ [웹 UI] → 사용자에게 표시
```

## 🌐 네트워크 포트

```
외부 접근:
  └─→ 포트 80: Apache 웹서버

내부 네트워크:
  ├─→ 포트 3306: MySQL (LAMP 내부)
  └─→ host.docker.internal:9001: NPU Worker

외부 API:
  └─→ HTTPS: 카카오 Map API
```
