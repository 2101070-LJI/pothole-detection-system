# 포트홀 합성 이미지 자동 생성 - 간단 실행 가이드

## 🚀 가장 간단한 실행 방법

### PowerShell 스크립트 사용 (권장)

```powershell
# 기본 실행 - 이것 하나면 끝!
.\generate_synthetic_potholes.ps1
```

이 명령어 하나로:
- ✅ 도로 이미지에서 포트홀 없는 이미지 자동 검색
- ✅ 각 이미지에 포트홀 자동 합성
- ✅ YOLO 형식 라벨 자동 생성
- ✅ 결과를 `/app/models/synthetic_dataset`에 저장

### 다양한 옵션 지정

```powershell
# 이미지당 포트홀 2개, 다양한 크기
.\generate_synthetic_potholes.ps1 -NumPotholes 2 -MinScale 0.3 -MaxScale 2.0
```

## 📋 실행 전 확인사항

1. **컨테이너 실행 확인**
   ```powershell
   docker ps --filter "name=deep-guardian-ai"
   ```
   컨테이너가 `Up` 상태여야 합니다.

2. **도로 이미지 준비**
   - `shared_images/detections` 디렉토리에 포트홀 없는 도로 이미지가 있어야 합니다
   - 지원 형식: `.jpg`, `.jpeg`, `.png`

## 🎯 실행 예시

### 예시 1: 기본 실행
```powershell
.\generate_synthetic_potholes.ps1
```
- 결과: 각 도로 이미지에 포트홀 1개씩 합성
- 크기: 0.5~1.5배 (기본값)

### 예시 2: 다양한 크기
```powershell
.\generate_synthetic_potholes.ps1 -NumPotholes 2 -MinScale 0.3 -MaxScale 2.0
```
- 결과: 각 이미지에 포트홀 2개씩 합성
- 크기: 0.3~2.0배 (더 다양한 크기)

### 예시 3: Docker 명령어 직접 사용
```powershell
docker exec deep-guardian-ai python /app/generate_synthetic_potholes.py --road_images /app/shared_images/detections --output_dir /app/models/synthetic_dataset --num_potholes 1
```

## 📂 결과 확인

생성된 파일 위치:
- **이미지**: `ai-core/models/synthetic_dataset/images/train/`
- **라벨**: `ai-core/models/synthetic_dataset/labels/train/`

각 라벨 파일 형식 (YOLO):
```
0 0.523456 0.345678 0.123456 0.098765
```
- `0`: 클래스 ID (포트홀)
- 다음 4개 숫자: 중심 X, 중심 Y, 너비, 높이 (0-1 범위)

## ⚠️ 문제 해결

### 컨테이너가 실행되지 않는 경우
```powershell
docker-compose up -d ai-core
```

### 권한 오류
```powershell
docker-compose restart ai-core
```

### 실행 후 확인
```powershell
# 생성된 이미지 확인
docker exec deep-guardian-ai ls /app/models/synthetic_dataset/images/train | Select-Object -First 5
```
